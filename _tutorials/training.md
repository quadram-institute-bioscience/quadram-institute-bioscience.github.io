We deliver **300+** hours of training every year, including formal classes, workshops, hackathons, open-houses and 1:1 training.

Here we list some regular activities that have a companion website:

## Technical

* [Bioinformatics from the Command Line](https://mmbdtp.github.io/modules/unix/week-1-overview/) - An introduction to surviving the Terminal interface and start running bioinformatics tools
* [Using the NBI-HPC](https://corebio.info/nbi-hpc/) - Slurm, Conda, Singularity. If you used a cluster before meet our cluster setup, if not let's have a quick intro to it

## Microbial genomics

* [Whole metagenome sequencing](https://mmbdtp.github.io/modules/metagenomics/programme/) - QC, host removal, taxonomic and functional classification, de novo assembly binning and some R code to make plots
* [Viromics workshop](https://corebio.info/ebame-viromics/) - Mine phages in metagenomes, derepliate and get a vOTU table (EBAME workshop)
